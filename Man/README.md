# TechNews / EmmaNews (Man Folder)

A responsive static news website that fetches live articles from NewsAPI and renders them as cards with category filtering and search.

## Overview

The `Man/` site is a **frontend-only** implementation:

- **HTML** for structure
- **CSS** for styling and responsive layout
- **JavaScript** for API calls and rendering

No backend service is required for local/demo usage.

---

## Features

- Real-time article fetching from NewsAPI
- Category filtering (General, Technology, Business, Science, Health)
- Keyword-based search
- Loading and error states
- Card-based article rendering with source/date metadata
- Responsive layout for desktop/tablet/mobile
- Basic client-side safety checks for missing fields
