# 🚀 QUICK START - ExpenseTracker

## ⚡ 5-Minute Setup

### 1️⃣ Start XAMPP
- Open XAMPP Control Panel
- Click **Start** → Apache
- Click **Start** → MySQL

### 2️⃣ Copy Project Files
1. Navigate to: `C:\xampp\htdocs\`
2. Create folder: `expense-tracker`
3. Copy all project files here

### 3️⃣ Initialize Database
Open browser → Visit:
```
http://localhost/expense-tracker/backend/db_init.php
```
✅ Should see: "Database initialization complete!"

### 4️⃣ Access Application
Open browser → Visit:
```
http://localhost/expense-tracker/
```

### 5️⃣ Create Account & Login
1. Click **Sign Up**
2. Fill form and create account
3. Click **Login** with credentials
4. Start tracking! 🎉

## 🛠️ Troubleshooting

### Database Error
- [ ] Ensure MySQL is running
- [ ] Check `backend/config.php`

### Cannot Login
- [ ] Verify email is registered
- [ ] Check password (case-sensitive)
- [ ] Clear browser cookies

### File Upload Fails
- [ ] File must be PDF/JPG/PNG
- [ ] File size must be < 5MB
- [ ] Check internet connection

### Page Not Loading
- [ ] Apache must be running
- [ ] Clear browser cache (Ctrl+Shift+Delete)
- [ ] Check URL spelling
